# Pull, Add, Commit, and Push Changes
Pull changes (to ensure your local repo is up to date):
git pull origin main

Add changes (stage your changes for commit):
git add .
or 
git add filename
or 
git rm filename 


Commit changes (save your staged changes with a message):
git commit -m "A descriptive message about your changes"

Push changes (upload your commits to GitHub):
git push origin main
